Communication Networks

MANET for D2D Networks

An ad-hoc mobile area network with variable number of base stations and mobile users with free space loss, multipath fading (Rayleigh fading), and interference was simulated in Python. When the base stations fail, a device-to-device(D2D) network is simulated over the existing infrastructure. Security in the form of pre-shared keys is introduced. The probability of forming a D2D connection and the overhead introduced by adding the security layer was calculated using the simulation.

Queueing in Load Balancers: A Comparison Between a Randomized Scheme and a Replication-Based Scheme

Two methods of load balancing – the supermarket model and the replication based model - are analyzed in detail and compared. The supermarket model chooses d servers from N and sends the job to the server with the minimum queue size. The replication based model sends replicas of the same job to d servers and removes the (d-1) replicas from other queues as soon as the job starts being processed by a server. The supermarket model is simulated and compared against the theoretical expected time spent in the system. The results show an improvement over randomized server selections. It was seen that the replication model reduces the overhead in the supermarket model even further.
